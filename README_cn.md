A Gateway built on Golang.

基于 Go 生态（包括：go、gin、zap、etc...）的网关。

## 特点
- Go 1.15
- Gin
- 动态路由
- 支持 HTTP、TCP、gRPC
- 请求上的路由匹配（路径，请求头，主机等...）
- 匹配到路由
- 过滤器可以修改下游请求和相求（添加/删除 请求头、参数，重写路径，设置路径，负载均衡，熔断等...）
